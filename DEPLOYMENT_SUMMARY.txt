================================================================================
DEPLOYMENT QUICK START
================================================================================

1. Update VPS and install Docker:
   ssh root@vps
   apt update && apt upgrade -y
   curl -fsSL https://get.docker.com | sh
   apt install -y docker-compose git

2. Clone and configure:
   git clone https://github.com/Aburtik3/1.git anonkeymail
   cd anonkeymail
   git checkout codex/set-up-vps-for-anonkeymail
   cp .env.example .env
   nano .env  # Fill in all values with secrets

3. Prepare SSL certificates:
   mkdir -p certs
   # Copy fullchain.pem and privkey.pem to certs/

4. Deploy:
   docker compose --profile production up -d --build

5. Verify:
   docker compose ps
   curl https://your-domain/api/health

For detailed instructions, see: QUICK_START_PROD.md
